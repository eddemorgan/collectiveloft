# Collective Loft Blog Posts

All blog posts live here as Markdown files.

## To publish a new post

1. Copy `template-for-contributors.md`
2. Rename it using your slug (e.g. `how-to-write-a-great-brief.md`)
3. Fill in all frontmatter fields
4. Write the post below the second `---`
5. Submit a PR to `main` — merging publishes the post immediately

## Frontmatter required fields

```yaml
title:     Full post title
slug:      url-safe-slug-matching-filename
date:      2026-06-15
author:    Your Name
category:  One of the 16 categories (see template)
layer:     1, 2, 3, or 4
featured:  false (only one post featured at a time)
excerpt:   One or two sentences shown on the index card.
```

## Optional fields

```yaml
cover: /blog/your-image.jpg   # place image in /public/blog/
```

## Cover images

Place cover images in `/public/blog/`. Recommended size: 1200×630px.
