## How to write a post

1. Copy this file and rename it using your post slug (e.g. `how-to-write-a-great-brief.md`)
2. Update every field in the frontmatter (the section between the --- lines at the top)
3. Write your post below the second `---`
4. Submit a PR to the main branch — the post publishes on merge

---

## Frontmatter field guide

```
title:     The full title of your post (shown on index and post page)
slug:      URL-safe version of the title, lowercase, hyphens (e.g. how-to-write-a-brief)
date:      Publication date in YYYY-MM-DD format
author:    Your name as you want it to appear
category:  Must be one of the 16 categories listed below
layer:     The layer number (1, 2, 3, or 4) — matches the category
featured:  true or false — only one post should be featured at a time
excerpt:   1-2 sentences shown on the index card. Write this last.
cover:     Optional. Path to image in /public/blog/ (e.g. /blog/my-image.jpg)
```

## Writing style guide

- Write like you talk to another creative. Not like a press release.
- Short paragraphs. One idea per paragraph.
- Use `##` for section headings. Use `**bold**` sparingly — only for things that actually matter.
- End with a one-line italicized note connecting back to the platform (see the sample post for the pattern).
- Target length: 600–1200 words. Shorter is fine. Longer only if the topic demands it.
- No jargon. No "synergy." No "leveraging."

## Image guide

- Place images in `/public/blog/`
- Name them clearly: `collab-terms-guide.jpg` not `IMG_4821.jpg`
- Recommended size: 1200×630px for cover images
- Add the path to the `cover:` field in frontmatter
